# ⛪ OrdoServus
**Der digitale Begleiter für Liturgie und Pastoral.**

OrdoServus ist eine Open-Source-Webanwendung, die speziell für Pfarrer, Diakone und pastorale Mitarbeiter entwickelt wurde. Sie kombiniert eine strukturierte Gottesdienst-Planung mit einem flexiblen Notizbuch.

## ✨ Funktionen
- **Liturgie-Planer:** Vorlagen für Messen, Taufen und Andachten.
- **Digitales Notizbuch:** Organisation in Registern (wie OneNote).
- **Cloud-Synchronisierung:** Sichere Anmeldung und Datenspeicherung in der Cloud, um von überall auf Ihre Daten zuzugreifen.
- **Export:** Einfacher PDF-Druck für den Dienst am Altar.

## 🚀 Technologie
- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** CSS-in-JS / Inline Styles
- **Authentifizierung & Datenbank:** Google Firebase

## 🛠️ Installation & Entwicklung
1. Repository klonen: `git clone https://github.com/flohulo/ordoservus.git`
2. Abhängigkeiten installieren: `npm install`
3. Lokal starten: `npm run dev`
